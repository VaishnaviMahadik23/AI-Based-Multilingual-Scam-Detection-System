# AI Scam Message Detector

## 🚀 Project Overview
An AI-powered multilingual scam detection system designed to identify fraudulent SMS, WhatsApp, and messages in real-time.

## 🎯 Features
- AI-based scam detection
- Multi-language support (English, Hindi, Marathi)
- Detects scam links and keywords
- Real-time risk scoring (Safe / Suspicious / Dangerous)
- Web-based UI

## 🧠 Tech Stack
- Python (Flask)
- Machine Learning (Naive Bayes)
- HTML, CSS, JavaScript
- Google Translate API

## 🔥 How it works
1. User enters message
2. System detects language
3. Converts to English
4. AI + rule-based scoring
5. Returns risk level + translated output

## 🌍 Use Case
Helps users (especially elders in India) avoid online fraud and phishing scams.
